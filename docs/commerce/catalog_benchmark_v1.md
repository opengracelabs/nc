# Catalog Benchmark v1.0

This benchmark defines the commercial catalog requirements for the core Nature & Culture asset fixtures. it codifies the variant structures, pricing strategies, and SEO standards required for deployment to Shopify and Etsy.

---

## 1. Price Bands & Size Ladders

We utilize three distinct pricing tiers based on the **Commercial Success Probability (CSP)** and **Classification**.

### 1.1 The Masterwork Tier (Premium)
*   **Target Assets:** Moran, Hayden Map.
*   **Media:** Archival Linen, Heavyweight Rag Paper (Giclée).
*   **Size Ladder:** 24x36", 36x48", 40x60" (Grand Scale).
*   **Price Band:** $150.00 — $1,200.00+.

### 1.2 The Flagship Tier (Standard)
*   **Target Assets:** Audubon, Haeckel, Jackson.
*   **Media:** Premium Matte Paper, Framed Canvas.
*   **Size Ladder:** 11x14", 16x20", 18x24", 24x36".
*   **Price Band:** $45.00 — $250.00.

### 1.3 The Heritage Tier (Volume)
*   **Target Assets:** Jackson, Redouté (Secondary).
*   **Media:** High-quality Poster Paper, Cards, Calendars.
*   **Size Ladder:** 8x10", 11x14", 12x12" (Calendars).
*   **Price Band:** $15.00 — $60.00.

## 3. Metadata & SEO Requirements

### 3.1 Title Patterns
*   **Pattern A (Premium):** `{Asset Title} ({Year}) — {Artist} — {Product Family}`
*   **Pattern B (Volume):** `{Asset Title} — Vintage {Place} {Product Family}`

### 3.2 Description Requirements (The "Curator's Note")
Every product description must include:
1.  **Provenance:** The source institution (e.g., "From the Geography and Map Division of the Library of Congress").
2.  **Scientific/Historical Context:** A 2-3 sentence "Story" (SSS).
3.  **Restoration Note:** "Digitally restored for modern display while preserving original historic patina."
4.  **Rights Attribution:** "Public Domain / CC0."

### 3.3 SEO Metafield Standards
*   **Alt Text:** Must describe the image subject, artist, and institutional source.
*   **Product Tags:** Must include `place:{place_id}`, `artist:{artist_id}`, `classification:{tier}`, `style:{style}`.
