# v0.4.0 Collection #000001

Collection #000001 adds governed commercial collections for verified public-domain BHL illustration assets.

## Added

- Collection schema: `collections`, `collection_places`, `collection_assets`, and `collection_exports`.
- Collection governance API: list/detail, approve, publish, reject, dispute, and retract.
- Collection asset safeguards requiring verified Public Domain or CC0 BHL illustration assets.
- Deterministic collection manifest export with checksum and export record metadata.

## Boundaries

- This release records manifest exports; it does not create commerce ZIP bundles.
- Collections consume ingested assets. They do not infer rights or duplicate BHL asset acquisition.
